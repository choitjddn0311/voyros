# voyros
의미있는 순간을 다른 시선으로 바라보다.

```
Voyros(보이로스)는 “의미 있는 순간”과 “다양한 시선”이 만나는 차세대 시각 커뮤니티입니다.
고대 그리스의 ‘카이로스’(결정적 순간)와 ‘보는 행위’(Voyr)에서 영감을 받아,
사진을 단순한 기록이 아닌 시간과 시선의 대화로 재해석합니다.
메타데이터와 위치 정보를 통해,
사용자는 사진을 올리는 것이 아니라, 세상을 바라보는 방식을 공유합니다.
```

1.  공유
모두 같은 세상을 살아도 다 다른 시선을 가지고 삽니다. 
정답 없는 이 시선을 함께 공유합니다.

2.  간직
순간을 간직합니다.
여러장의 사진보다 하나의 의미를 가지고 찍습니다.
